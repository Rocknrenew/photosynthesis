word.camera uses convolutional neural networks (via Clarifai) to extract concept words from images. It expands those initial words (mostly nouns) into sentences and paragraphs using a lexical relationship database (ConceptNet) and a flexible template system.

word.camera should work on most mobile devices with modern browsers, though page load time remains an issue. Along with that, I am currently working on improving word.camera's grammar and introducing some new features. This project is open source, and all my code is available on GitHub.
