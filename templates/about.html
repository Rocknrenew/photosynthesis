word.camera uses convolutional neural networks (via Clarifai) to extract concept words from images. It expands those initial words (mostly nouns) into sentences and paragraphs using a lexical relationship database (ConceptNet) and a flexible template system.

This project is currently under active development. At the moment, word.camera should work in recent versions of Chrome, Firefox, and on most mobile devices with modern browsers, though processing time remains an issue. Along with that, improving word.camera's grammar remains the main development priority. This project is open source, and its code is available on GitHub.

Ross Goodwin's blog is thehypertext.com. To receive updates about this particular project, please sign up for the Lexography newsletter below. (One email per week at most, and likely far less frequent than that.)
